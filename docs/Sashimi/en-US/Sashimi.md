# Sashimi Module

## Description

Raw process I/O for PowerShell — execute external commands and handle stdin/stdout/stderr as byte streams.

## Sashimi

### [ConvertFrom-String](ConvertFrom-String.md)

Converts a PowerShell string into a byte array using the specified text encoding.

### [ConvertTo-String](ConvertTo-String.md)

Converts raw byte input into a PowerShell string using the specified text encoding.

### [Invoke-RawCommand](Invoke-RawCommand.md)

Executes a native command and returns its raw byte output without any text encoding or PowerShell string conversion.
